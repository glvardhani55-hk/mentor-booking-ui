UI Checklist
  • Responsive grid: mobile → tablet → desktop
  • Reusable components: cards, chips, badges
  • Accessibility: visible focus + readable contrast
  • Forms: validation messages + required fields
  • Bootstrap components: navbar, modal, accordion
